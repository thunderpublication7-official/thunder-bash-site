How to install debian proot-distro of your choice ==>

pkg update -y

pkg upgrade -y

pkg install wget -y

wget https://raw.githubusercontent.com/thunderpublication7-official/thunderbash/refs/heads/main/script.sh

chmod +x script.sh

./script.sh

After login you will see root@localhost:~# type following copy paste following commands ==>

apt update -y

apt upgrade -y

apt install wget -y

wget https://raw.githubusercontent.com/thunderpublication7-official/thunderbash/refs/heads/main/script2.sh

chmod +x script2.sh

./script2.sh

Now you will see nano editor remove all content in file and add following lines ==>

#!/bin/sh
unset SESSION_MANAGER
unset DBUS_SESSION_BUS_ADDRESS
xrdb $HOME/.Xresources
startxfce4 &

After pasting lines in nano editor press 'ctrl + x' then it will ask you to save changes press 'y' to save changes then press 'enter' to exit nano editor.

After that copy and paste following lines in termimal ==>

chmod +x ~/.vnc/xstartup

vncserver :1

Now on typing vncserver :1 vnc server is started on port 1. to join server download any vnc veiwer, and click on + icon and fill details in address type 'localhost:1' and fill any name of your choice.


now to stop server type this code ==>

vncserver -kill :1

now to quit debian session from terminal type ==>

quit

to start server again type this in terminal session to login in debian followed by command ahead==>

pd login debian

to start server type this command in debian session in terminal ==>

vncserver :1
